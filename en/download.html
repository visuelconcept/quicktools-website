Requirements

  • Windows 10 or Windows 11
  • Visual Studio 2015 or later (for development)
  • Internet connection for license activation (offline activation available)

License activation

Activate your license in a few simple steps with QuickActivator.

Install QuickTOOLS

Run the installer and follow the instructions. The installer automatically offers to activate a free trial.

Activate a trial (automatic)

On first launch, a trial license is automatically activated. You can also run manually:

QuickActivator.exe trial

Activate a full license

Once you receive your license key, activate it with:

QuickActivator.exe activate YOUR-LICENSE-KEY

Check status

Check your license status at any time with:

QuickActivator.exe status

Offline activation

For air-gapped environments, QuickTOOLS supports offline activation:

  1. Generate a request file: QuickActivator.exe offline-activate-request
  2. Transfer this file to a machine with internet access.
  3. Apply the response: QuickActivator.exe offline-activate-response

See the full documentation for details.
