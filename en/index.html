Three pillars, one solution

QuickTOOLS brings together three complementary components for communication, synchronization and persistence.

QuickSERVER

Publish/subscribe message broker for real-time communication between applications.

  • Datagroups for message routing
  • Compression and load balancing
  • Typed messages with binary serialization
  • Built-in HTTP dashboard
  • Automatic client reconnection
Learn more →

QuickSDB

Real-time shared database with transparent synchronization across all clients.

  • Synchronized typed variables
  • Object model with classes and inheritance
  • Computed algebraic formulas
  • Disk persistence
  • Transaction support
Learn more →

DbEngine

Unified SQL abstraction layer for 6 database engines.

  • SQLite (embedded, default)
  • MySQL and PostgreSQL
  • Oracle and MS SQL Server
  • Sybase
  • Switch engines without recompilation
Learn more →
