Lizenzverwaltung durch Cryptlex

Cryptlex ist eine sichere cloudbasierte Plattform zur Verwaltung von Softwarelizenzen. Sie übernimmt die Aktivierung, Validierung und Nachverfolgung der QuickTOOLS-Lizenzen, mit Unterstützung für Online- und Offline-Aktivierung. Die Lizenzdaten werden in der Windows-Registry gespeichert — keine Lizenzdatei erforderlich.

Lizenzaktivierung

Aktivieren Sie Ihre Lizenz in wenigen einfachen Schritten mit QuickActivator.

QuickTOOLS installieren

Führen Sie das Installationsprogramm aus und folgen Sie den Anweisungen. Das Installationsprogramm bietet automatisch die Aktivierung einer kostenlosen Testversion an.

Testversion aktivieren (automatisch)

Beim ersten Start wird automatisch eine Testlizenz aktiviert. Sie können auch manuell starten:

QuickActivator.exe trial

Volllizenz aktivieren

Sobald Sie Ihren Lizenzschlüssel erhalten haben, aktivieren Sie ihn mit:

QuickActivator.exe activate IHR-LIZENZSCHLUESSEL

Status prüfen

Überprüfen Sie Ihren Lizenzstatus jederzeit mit:

QuickActivator.exe status

Offline-Aktivierung

Für Umgebungen ohne Internetzugang unterstützt QuickTOOLS die Offline-Aktivierung:

  1. Generieren Sie eine Anfragedatei: QuickActivator.exe offline-activate-request
  2. Übertragen Sie diese Datei auf einen Computer mit Internetzugang.
  3. Wenden Sie die Antwort an: QuickActivator.exe offline-activate-response

Weitere Details finden Sie in der vollständigen Dokumentation.
