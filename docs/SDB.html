QuickSDB — API de base de donnees partagee

Presentation

QuickSDB est une base de donnees distribuee de variables partagees fonctionnant au-dessus de QuickSERVER. Elle assure la synchronisation en temps reel de variables typees entre les clients connectes au reseau.

Fonctionnalites principales :

  • Variables typees partagees (booleens, entiers, flottants, chaines, dates, etc.)
  • Synchronisation automatique en temps reel entre tous les clients
  • Notifications de changement par variable ou par joker
  • Classes et objets pour structurer les donnees
  • Enumerations personnalisees
  • Formules conditionnelles avec evaluation automatique
  • Groupes de variables pour les operations par lot
  • Alias (indirection de variables)
  • Transactions avec commit et rollback

Exemple complet

Le programme suivant cree une variable de temperature, surveille ses modifications et ecrit une valeur :

#include "QT_std.h"
#include "QT_socket.h"
#include "QT_sdb.h"

void on_change(T_SDB inst, char *var, E_FIELD type, T_FIELD_VALUE *val, void *arg)
{
    printf("Variable %s changee: %.2f\n", var, val->Float);
}

int main(int argc, char *argv[])
{
    QT_Init(argc, argv);
    QT_SocketInit("localhost", "client_sdb", C_AUTO_RECONNECTION_CONTEXT);

    /* Connexion a l'instance 1 */
    T_DATAGROUP dg = QT_SdbConnect("localhost", 1);

    /* Lecture de la structure existante */
    QT_SdbReadStructure(1);

    /* Ajout d'une variable de type double */
    QT_SdbAddVariable(1, "TEMPERATURE", C_DOUBLE, 0.0);

    /* Surveillance des modifications */
    QT_SdbWatchVariable(1, "TEMPERATURE", on_change, NULL);

    /* Ecriture d'une valeur */
    QT_SdbSetDouble(1, "TEMPERATURE", 42.5);

    /* Validation des modifications */
    QT_SdbCommit(1);

    /* Attente des messages pendant 30 secondes */
    QT_SocketWaitMsg(0, 30);

    /* Deconnexion */
    QT_SdbDisconnect(1);
    QT_SocketTerminate();

    return 0;
}
