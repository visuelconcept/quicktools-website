UTIL — Fonctions utilitaires et structures de donnees

Presentation

Le module UTIL constitue le socle fondamental de la bibliotheque QuickTOOLS. Il fournit un ensemble complet de fonctions utilitaires couvrant la gestion memoire, la manipulation de chaines de caracteres, les operations sur les dates et heures, la configuration, la compression, le chargement dynamique de bibliotheques, l'internationalisation, ainsi que trois structures de donnees generiques : la file d'attente (queue), la table de symboles (symtable) et la table de hachage (hashtable).

En-tetes : - QT_std.h — Types standard et macros de base - QT_util.h — Fonctions utilitaires principales - queue.h — File d'attente generique - symtable.h — Table de symboles (arbre binaire equilibre) - hashtable.h — Table de hachage - license.h — Gestion des licences

Bibliotheque : qt_util.lib

2. Initialisation

QT_Init

void QT_Init(int argc, char *argv[]);

Initialise le framework QuickTOOLS. Cette fonction doit etre appelee avant toute autre fonction QuickTOOLS. Elle analyse les arguments de la ligne de commande et initialise les structures internes (memoire, configuration, journalisation).

Parametres : - argc — Nombre d'arguments de la ligne de commande - argv — Tableau des arguments

QT_Terminate

void QT_Terminate(long status);

Termine proprement le framework QuickTOOLS et quitte le programme. Libere toutes les ressources internes, ferme les fichiers ouverts et appelle exit(status).

Parametres : - status — Code de retour du programme (0 = succes)

3. Gestion memoire

Fonctions standard

QT_Free

void QT_Free(void *Addr);

Libere un bloc memoire precedemment alloue par QT_Malloc ou QT_Realloc. Encapsule free() avec suivi optionnel en mode debug.

Parametres : - Addr — Pointeur vers le bloc memoire a liberer

QT_Malloc

void *QT_Malloc(size_t Size);

Alloue un bloc memoire de la taille specifiee. Encapsule malloc() avec verification d'erreur et suivi optionnel en mode debug.

Parametres : - Size — Taille en octets du bloc a allouer

Retour : - Pointeur vers le bloc alloue, ou NULL en cas d'echec

QT_Realloc

void *QT_Realloc(void *Addr, size_t Size);

Realloue un bloc memoire existant avec une nouvelle taille. Le contenu est preserve jusqu'a la taille minimale entre l'ancienne et la nouvelle.

Parametres : - Addr — Pointeur vers le bloc existant - Size — Nouvelle taille en octets

Retour : - Pointeur vers le bloc realloue, ou NULL en cas d'echec

Fonctions de debug memoire

Ces fonctions ne sont disponibles que lorsque le programme est compile avec l'indicateur DEBUG. En mode release, elles sont remplacees par les fonctions standard via des macros.

QT_dbgMalloc

void *QT_dbgMalloc(char *File, size_t Size);

Version debug de malloc. Enregistre le fichier source et la ligne d'appel pour le suivi des allocations.

Parametres : - File — Fichier source et numero de ligne (rempli automatiquement par la macro) - Size — Taille en octets a allouer

Retour : - Pointeur vers le bloc alloue

QT_dbgFree

void QT_dbgFree(char *File, void *Addr);

Version debug de free. Verifie la validite du pointeur et met a jour le suivi des allocations.

Parametres : - File — Fichier source et numero de ligne - Addr — Pointeur vers le bloc a liberer

QT_dbgRealloc

void *QT_dbgRealloc(char *File, void *Addr, size_t Size);

Version debug de realloc. Enregistre la reallocation pour le suivi.

Parametres : - File — Fichier source et numero de ligne - Addr — Pointeur vers le bloc existant - Size — Nouvelle taille en octets

Retour : - Pointeur vers le bloc realloue

QT_dbgReportLeaks

void QT_dbgReportLeaks(void);

Affiche un rapport de toutes les fuites memoire detectees depuis le dernier appel a QT_dbgMark. Chaque fuite est identifiee par le fichier source, la ligne et la taille du bloc non libere.

QT_dbgMemStats

void QT_dbgMemStats(void);

Affiche les statistiques memoire : nombre total d'allocations, de liberations, taille totale allouee, pic d'utilisation memoire.

QT_dbgMark

void QT_dbgMark(void);

Marque l'etat courant de la memoire. Les fuites detectees par QT_dbgReportLeaks seront relatives a ce point de reference.

QT_dbgStopHere

void QT_dbgStopHere(void);

Fonction vide destinee a poser un point d'arret dans le debogueur. Appelee automatiquement lorsqu'une erreur memoire est detectee.

QT_dbgFindBad

int QT_dbgFindBad(void);

Recherche les blocs memoire corrompus (depassement de tampon, ecriture hors limites).

Retour : - Nombre de blocs corrompus trouves

QT_dbgMarkChanges

void QT_dbgMarkChanges(void);

Enregistre un instantane de l'etat memoire pour le suivi des modifications.

QT_dbgFindChanges

void QT_dbgFindChanges(void);

Affiche les modifications memoire survenues depuis le dernier appel a QT_dbgMarkChanges.

QT_dbgReportAllocs

long QT_dbgReportAllocs(void);

Affiche la liste de toutes les allocations en cours.

Retour : - Nombre d'allocations actives

QT_dbgReportSince

int QT_dbgReportSince(int Id, int Size);

Affiche les allocations effectuees depuis un identifiant ou une taille donnee.

Parametres : - Id — Identifiant d'allocation de reference - Size — Taille d'allocation de reference

Retour : - Nombre d'allocations rapportees

QT_dbgIdStop

void QT_dbgIdStop(long Id);

Definit un point d'arret sur un identifiant d'allocation. Lorsque l'allocation portant cet identifiant sera effectuee, QT_dbgStopHere sera appelee.

Parametres : - Id — Identifiant d'allocation cible

QT_dbgSizeStop

void QT_dbgSizeStop(size_t Size);

Definit un point d'arret sur une taille d'allocation. Toute allocation de cette taille exacte declenchera QT_dbgStopHere.

Parametres : - Size — Taille d'allocation cible

QT_dbgHasError

boolean QT_dbgHasError(void);

Verifie si des erreurs memoire ont ete detectees (corruption, double liberation, etc.).

Retour : - C_YES si des erreurs ont ete detectees - C_NO sinon

4. Gestion d'erreurs

E_ERROR_GRAVITY

Enumeration definissant la gravite d'une erreur.

Constante Description
C_WARNING Avertissement — le programme continue normalement
C_SEVERE Erreur grave — le programme peut continuer mais le comportement est potentiellement degrade
C_FATAL Erreur fatale — le programme se termine immediatement

QT_Error

void QT_Error(E_ERROR_GRAVITY type, char *fcn_name, char *msg, ...);

Signale une erreur. Le message est formate a la maniere de printf. En cas d'erreur fatale (C_FATAL), le programme est termine apres affichage du message.

Parametres : - type — Gravite de l'erreur - fcn_name — Nom de la fonction dans laquelle l'erreur s'est produite - msg — Message au format printf - ... — Arguments variables du message

Exemple :

QT_Error(C_FATAL, "MaFonction", "Fichier '%s' introuvable", nomFichier);

5. Chaines de caracteres

QT_StringUpper

char *QT_StringUpper(char *String);

Convertit une chaine de caracteres en majuscules, sur place.

Parametres : - String — Chaine a convertir (modifiee directement)

Retour : - Pointeur vers la chaine modifiee

QT_StringLower

char *QT_StringLower(char *String);

Convertit une chaine de caracteres en minuscules, sur place.

Parametres : - String — Chaine a convertir (modifiee directement)

Retour : - Pointeur vers la chaine modifiee

QT_StringClone

char *QT_StringClone(char *String);

Duplique une chaine de caracteres. Alloue la memoire necessaire et copie le contenu. L'appelant est responsable de liberer la memoire avec QT_Free.

Parametres : - String — Chaine a dupliquer

Retour : - Pointeur vers la nouvelle copie, ou NULL en cas d'echec

QT_StringExprCompare

boolean QT_StringExprCompare(char *String, char *Filter, boolean IgnoreCase);

Compare une chaine a un motif contenant des caracteres generiques (wildcards). Le caractere * remplace zero ou plusieurs caracteres quelconques ; le caractere ? remplace exactement un caractere.

Parametres : - String — Chaine a tester - Filter — Motif de comparaison (supporte * et ?) - IgnoreCase — C_YES pour ignorer la casse, C_NO pour une comparaison stricte

Retour : - C_YES si la chaine correspond au motif - C_NO sinon

Exemple :

QT_StringExprCompare("rapport_2026.pdf", "*.pdf", C_YES);   /* C_YES */
QT_StringExprCompare("image.png", "*.jp?g", C_NO);          /* C_NO  */

QT_StringSplit

int QT_StringSplit(char *List, char **ArgValue[], char Separator);

Decoupe une chaine en un tableau de sous-chaines selon un caractere separateur. La chaine d'origine est modifiee (les separateurs sont remplaces par '\0'). Le tableau de pointeurs est alloue en interne.

Parametres : - List — Chaine a decouper (modifiee sur place) - ArgValue — Pointeur vers le tableau de resultats (alloue par la fonction) - Separator — Caractere separateur

Retour : - Nombre de sous-chaines extraites

Exemple :

char liste[] = "un,deux,trois";
char **valeurs;
int n = QT_StringSplit(liste, &valeurs, ',');
/* n == 3, valeurs[0]=="un", valeurs[1]=="deux", valeurs[2]=="trois" */

QT_StringAppend

char *QT_StringAppend(char *Buffer, const char *String, ...);

Ajoute une chaine formatee (style printf) a la fin d'un tampon dynamique. Le tampon est automatiquement realloue si necessaire. Si Buffer est NULL, un nouveau tampon est cree.

Parametres : - Buffer — Tampon de destination (peut etre NULL) - String — Chaine de format (style printf) - ... — Arguments variables

Retour : - Pointeur vers le tampon (eventuellement realloue)

Exemple :

char *buf = NULL;
buf = QT_StringAppend(buf, "Bonjour %s, ", nom);
buf = QT_StringAppend(buf, "vous avez %d messages.", count);
/* buf == "Bonjour Jean, vous avez 5 messages." */
QT_Free(buf);

QT_StringInsert

char *QT_StringInsert(char *Buffer, unsigned long Position, const char *String, ...);

Insere une chaine formatee (style printf) dans un tampon a la position specifiee. Le tampon est automatiquement realloue si necessaire.

Parametres : - Buffer — Tampon de destination - Position — Position d'insertion (indice dans la chaine) - String — Chaine de format (style printf) - ... — Arguments variables

Retour : - Pointeur vers le tampon (eventuellement realloue)

QT_StringCat

char *QT_StringCat(char *Buffer, const char *String);

Concatene une chaine a la fin d'un tampon dynamique. Similaire a QT_StringAppend mais sans formatage. Le tampon est automatiquement realloue si necessaire.

Parametres : - Buffer — Tampon de destination (peut etre NULL) - String — Chaine a concatener

Retour : - Pointeur vers le tampon (eventuellement realloue)

6. Dates et heures

E_INTERVAL

Enumeration des intervalles de temps utilises par les fonctions de date.

Constante Description
C_YEAR Annee
C_QUARTER Trimestre
C_MONTH Mois
C_DAY_OF_YEAR Jour de l'annee (1-366)
C_DAY Jour
C_WEEK Semaine
C_WEEKDAY Jour de la semaine (0=dimanche, 6=samedi)
C_HOUR Heure
C_MINUTE Minute
C_SECOND Seconde

QT_DateGMTOffset

int QT_DateGMTOffset(void);

Retourne le decalage horaire entre l'heure locale et le temps universel coordonne (GMT/UTC), en secondes.

Retour : - Decalage en secondes (positif a l'est de Greenwich, negatif a l'ouest)

Exemple :

int offset = QT_DateGMTOffset();
/* En France (CET) : offset == 3600 (1 heure) */

QT_DateCreate

time_t QT_DateCreate(int Day, int Month, int Year, int Hour, int Min, int Sec, boolean GMT);

Cree une valeur time_t a partir de composantes de date et d'heure.

Parametres : - Day — Jour du mois (1-31) - Month — Mois (1-12) - Year — Annee complete (ex : 2026) - Hour — Heure (0-23) - Min — Minute (0-59) - Sec — Seconde (0-59) - GMT — C_YES pour interpreter les valeurs en GMT, C_NO pour l'heure locale

Retour : - Valeur time_t correspondante

Exemple :

time_t noel = QT_DateCreate(25, 12, 2026, 0, 0, 0, C_NO);

QT_DatePrint

char *QT_DatePrint(char *Format, struct tm *tp, time_t DateGMT);

Formate une date/heure en chaine de caracteres selon un format de type strftime.

Parametres : - Format — Chaine de format (codes strftime : %Y, %m, %d, %H, %M, %S, etc.) - tp — Pointeur vers une structure tm (peut etre NULL si DateGMT est utilise) - DateGMT — Valeur time_t (utilisee si tp est NULL)

Retour : - Chaine formatee (tampon interne statique)

Codes de format courants :

Code Description Exemple
%Y Annee sur 4 chiffres 2026
%m Mois (01-12) 03
%d Jour (01-31) 08
%H Heure (00-23) 14
%M Minute (00-59) 30
%S Seconde (00-59) 00

Exemple :

char *s = QT_DatePrint("%d/%m/%Y %H:%M:%S", NULL, time(NULL));
/* s == "08/03/2026 14:30:00" */

QT_DateScan

char *QT_DateScan(char *Format, char *Date, struct tm *tp);

Analyse une chaine de caracteres representant une date selon un format donne et remplit une structure tm.

Parametres : - Format — Chaine de format (style strftime) - Date — Chaine de date a analyser - tp — Pointeur vers la structure tm a remplir

Retour : - Pointeur vers la portion non analysee de la chaine - NULL en cas d'erreur d'analyse

Exemple :

struct tm tp;
QT_DateScan("%d/%m/%Y", "25/12/2026", &tp);
/* tp.tm_mday==25, tp.tm_mon==11 (0-base), tp.tm_year==126 */

QT_DateAdd

time_t QT_DateAdd(E_INTERVAL Interval, long Number, time_t DateGMT);

Ajoute un nombre d'intervalles a une date. Le nombre peut etre negatif pour soustraire.

Parametres : - Interval — Type d'intervalle (C_YEAR, C_MONTH, C_DAY, C_HOUR, C_MINUTE, C_SECOND, etc.) - Number — Nombre d'intervalles a ajouter (negatif pour soustraire) - DateGMT — Date de depart

Retour : - Nouvelle date

Exemple :

time_t demain = QT_DateAdd(C_DAY, 1, time(NULL));
time_t il_y_a_3_mois = QT_DateAdd(C_MONTH, -3, time(NULL));

QT_DateDiff

long QT_DateDiff(E_INTERVAL Interval, time_t DateGMT1, time_t DateGMT2);

Calcule la difference entre deux dates, exprimee dans l'unite specifiee.

Parametres : - Interval — Unite du resultat (C_DAY, C_HOUR, C_MINUTE, C_SECOND, etc.) - DateGMT1 — Premiere date - DateGMT2 — Seconde date

Retour : - Difference exprimee dans l'unite demandee (DateGMT2 - DateGMT1)

Exemple :

long jours = QT_DateDiff(C_DAY, dateDebut, dateFin);

QT_DateTruncate

time_t QT_DateTruncate(time_t DateGMT, E_INTERVAL Interval);

Tronque une date a la borne inferieure de l'intervalle specifie. Par exemple, tronquer au jour met l'heure, les minutes et les secondes a zero.

Parametres : - DateGMT — Date a tronquer - Interval — Intervalle de troncature (C_YEAR, C_MONTH, C_DAY, C_HOUR, C_MINUTE)

Retour : - Date tronquee

Exemple :

time_t debut_jour = QT_DateTruncate(time(NULL), C_DAY);
time_t debut_mois = QT_DateTruncate(time(NULL), C_MONTH);

QT_DatePart

long QT_DatePart(E_INTERVAL Interval, time_t DateGMT);

Extrait une composante d'une date.

Parametres : - Interval — Composante a extraire (C_YEAR, C_MONTH, C_DAY, C_HOUR, C_MINUTE, C_SECOND, C_WEEKDAY, C_DAY_OF_YEAR) - DateGMT — Date source

Retour : - Valeur de la composante demandee

Exemple :

long annee = QT_DatePart(C_YEAR, time(NULL));     /* 2026 */
long mois  = QT_DatePart(C_MONTH, time(NULL));    /* 1-12 */
long jour  = QT_DatePart(C_WEEKDAY, time(NULL));  /* 0=dim, 6=sam */

Exemple complet

L'exemple suivant illustre l'utilisation des principales fonctions du module UTIL : initialisation, lecture de configuration, manipulation de fichiers INI, operations sur les chaines et gestion des dates.

#include "QT_std.h"
#include "QT_util.h"

int main(int argc, char *argv[])
{
    /* ---------------------------------------------- */
    /* 1. Initialisation de QuickTOOLS                */
    /* ---------------------------------------------- */
    QT_Init(argc, argv);

    /* Afficher la version */
    long maj, min, rev;
    _QuickVersionNumber(&maj, &min, &rev);
    printf("QuickTOOLS %ld.%ld.%ld\n", maj, min, rev);

    /* ---------------------------------------------- */
    /* 2. Lecture du fichier de configuration          */
    /* ---------------------------------------------- */
    if (QT_ReadConfigFile("monappli") == C_YES) {
        char *serveur = QT_GetConfigKey("BaseDeDonnees", "Serveur");
        long  port    = QT_GetPrivateProfileInt("BaseDeDonnees", "Port", 3306);
        char *login   = QT_GetPrivateProfileString("BaseDeDonnees", "Login", "root");

        printf("Connexion a %s:%ld en tant que %s\n", serveur, port, login);
        QT_CloseConfig();
    }

    /* ---------------------------------------------- */
    /* 3. Manipulation d'un fichier INI               */
    /* ---------------------------------------------- */
    T_INIFILE *ini = QT_IniParseFile("parametres.ini");
    if (ini == NULL) {
        ini = QT_IniCreate();
    }

    /* Lire des valeurs avec des valeurs par defaut */
    char  *theme    = QT_IniGetString(ini, "Interface", "Theme", "clair");
    long   largeur  = QT_IniGetInteger(ini, "Fenetre", "Largeur", 1024);
    double zoom     = QT_IniGetDouble(ini, "Fenetre", "Zoom", 1.0);

    printf("Theme : %s, Largeur : %ld, Zoom : %.1f\n", theme, largeur, zoom);

    /* Modifier et sauvegarder */
    QT_IniSetString(ini, "Interface", "Theme", "sombre");
    QT_IniSetInteger(ini, "Fenetre", "Largeur", 1920);
    QT_IniSetDouble(ini, "Fenetre", "Zoom", 1.5);
    QT_IniWriteFile(ini, "parametres.ini");
    QT_IniDestroy(ini);

    /* ---------------------------------------------- */
    /* 4. Operations sur les chaines de caracteres    */
    /* ---------------------------------------------- */
    /* Cloner et convertir en majuscules */
    char *nom = QT_StringClone("Jean Dupont");
    QT_StringUpper(nom);
    printf("Majuscules : %s\n", nom); /* JEAN DUPONT */

    /* Construction dynamique d'une chaine */
    char *requete = NULL;
    requete = QT_StringAppend(requete, "SELECT * FROM %s", "clients");
    requete = QT_StringAppend(requete, " WHERE ville = '%s'", "Paris");
    requete = QT_StringAppend(requete, " ORDER BY %s", "nom");
    printf("Requete : %s\n", requete);

    /* Comparaison par motif */
    if (QT_StringExprCompare("rapport_2026.pdf", "rapport_*.pdf", C_NO)) {
        printf("Le fichier correspond au motif\n");
    }

    /* Decoupage d'une chaine */
    char liste[] = "rouge;vert;bleu;jaune";
    char **couleurs;
    int nb = QT_StringSplit(liste, &couleurs, ';');
    for (int i = 0; i < nb; i++) {
        printf("  Couleur %d : %s\n", i + 1, couleurs[i]);
    }

    QT_Free(nom);
    QT_Free(requete);

    /* ---------------------------------------------- */
    /* 5. Operations sur les dates                    */
    /* ---------------------------------------------- */
    /* Date courante */
    time_t maintenant = time(NULL);
    printf("Aujourd'hui : %s\n",
           QT_DatePrint("%d/%m/%Y %H:%M:%S", NULL, maintenant));

    /* Creer une date specifique */
    time_t noel = QT_DateCreate(25, 12, 2026, 0, 0, 0, C_NO);
    printf("Noel : %s\n", QT_DatePrint("%A %d %B %Y", NULL, noel));

    /* Calculer la difference en jours */
    long jours_avant_noel = QT_DateDiff(C_DAY, maintenant, noel);
    printf("Jours avant Noel : %ld\n", jours_avant_noel);

    /* Ajouter 90 jours */
    time_t dans_90_jours = QT_DateAdd(C_DAY, 90, maintenant);
    printf("Dans 90 jours : %s\n",
           QT_DatePrint("%d/%m/%Y", NULL, dans_90_jours));

    /* Tronquer au debut du mois */
    time_t debut_mois = QT_DateTruncate(maintenant, C_MONTH);
    printf("Debut du mois : %s\n",
           QT_DatePrint("%d/%m/%Y", NULL, debut_mois));

    /* Extraire des composantes */
    printf("Annee : %ld, Mois : %ld, Jour : %ld\n",
           QT_DatePart(C_YEAR, maintenant),
           QT_DatePart(C_MONTH, maintenant),
           QT_DatePart(C_DAY, maintenant));

    /* ---------------------------------------------- */
    /* 6. Terminaison                                 */
    /* ---------------------------------------------- */
    QT_Terminate(0);
    return 0;
}
