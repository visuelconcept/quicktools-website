Tre pilastri, una soluzione

QuickTOOLS riunisce tre componenti complementari per comunicazione, sincronizzazione e persistenza.

QuickSERVER

Message broker publish/subscribe per la comunicazione in tempo reale tra applicazioni.

  • Datagroup per l'instradamento dei messaggi
  • Compressione e bilanciamento del carico
  • Messaggi tipizzati con serializzazione binaria
  • Dashboard HTTP integrato
  • Riconnessione automatica dei client
Scopri di più →

QuickSDB

Database condiviso in tempo reale con sincronizzazione trasparente.

  • Variabili tipizzate sincronizzate
  • Modello a oggetti con classi ed ereditarietà
  • Formule algebriche calcolate
  • Persistenza su disco
  • Supporto transazionale
Scopri di più →

DbEngine

Livello di astrazione SQL unificato per 6 engine di database.

  • SQLite (integrato, predefinito)
  • MySQL e PostgreSQL
  • Oracle e MS SQL Server
  • Sybase
  • Cambio engine senza ricompilazione
