Prerequisiti

  • Windows 10 o Windows 11
  • Visual Studio 2015 o successivo (per lo sviluppo)
  • Connessione internet per l'attivazione della licenza (attivazione offline disponibile)

Attivazione della licenza

Attivate la vostra licenza in pochi semplici passaggi con QuickActivator.

Installare QuickTOOLS

Eseguite il programma di installazione e seguite le istruzioni. Il programma di installazione propone automaticamente l'attivazione di una prova gratuita.

Attivare una prova (automatica)

Al primo avvio, una licenza di prova viene attivata automaticamente. Potete anche avviare manualmente:

QuickActivator.exe trial

Attivare una licenza completa

Una volta ricevuta la chiave di licenza, attivatela con:

QuickActivator.exe activate VOSTRA-CHIAVE-DI-LICENZA

Verificare lo stato

Controllate lo stato della vostra licenza in qualsiasi momento con:

QuickActivator.exe status

Attivazione offline

Per ambienti senza accesso a internet, QuickTOOLS supporta l'attivazione offline:

  1. Generate un file di richiesta: QuickActivator.exe offline-activate-request
  2. Trasferite questo file su un computer con accesso a internet.
  3. Applicate la risposta: QuickActivator.exe offline-activate-response

Consultate la documentazione completa per i dettagli.
