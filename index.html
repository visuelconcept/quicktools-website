Trois piliers, une solution

QuickTOOLS réunit trois composants complémentaires pour répondre à vos besoins de communication, synchronisation et persistance.

QuickSERVER

Message broker publish/subscribe pour la communication temps réel entre applications.

  • Datagroups pour le routage des messages
  • Compression et load balancing
  • Messages typés avec sérialisation binaire
  • Dashboard HTTP intégré
  • Reconnexion automatique des clients
En savoir plus →

QuickSDB

Base de données partagée en temps réel avec synchronisation transparente.

  • Variables typées synchronisées
  • Modèle objet avec classes et héritage
  • Formules algébriques calculées
  • Persistance sur disque
  • Support transactionnel
En savoir plus →

DbEngine

Couche d'abstraction SQL unifiée pour 6 moteurs de bases de données.

  • SQLite (embarqué, par défaut)
  • MySQL et PostgreSQL
  • Oracle et MS SQL Server
  • Sybase
  • Changement de moteur sans recompilation
En savoir plus →
